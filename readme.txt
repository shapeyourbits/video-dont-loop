Removes the loop attribute from videos inside a core Cover block when the block has "noloop" CSS class in the Additional CSS class(es).

== Description ==

By default, videos inside WordPress Cover blocks loop endlessly. Video Don't Loop gives you a simple way to stop that — just add the CSS class `noloop` to any Cover block and the video will play once and stop.

**How to use:**

1. Edit a page or post in the WordPress block editor.
2. Select a Cover block that contains a video.
3. In the block settings sidebar, scroll down to "Advanced" and add `noloop` to the "Additional CSS class(es)" field.
4. Save or publish — the video will no longer loop.

No settings page, no configuration needed. Lightweight and focused on doing one thing well.

== Installation ==

= To install the plugin automatically: =

* Through WordPress admin, use the menu: Plugin > Add new
* Search for Video Don't Loop
* Click on install then click activate link

= To install the plugin manually: =

* Download and unzip the plugin video-dont-loop.zip
* Upload /video-dont-loop/ to the /wp-content/plugins/ directory
* Activate the plugin through the 'Plugins' screen in WordPress
* Refer to the How to Use section to turn off looping for a specific Cover block.
